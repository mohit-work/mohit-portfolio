## 🚀 About

This is my personal portfolio website — a high-fidelity, interactive showcase of my engineering projects, technical skills, and professional background. Built with a premium glassmorphism aesthetic, fluid animations, and an interactive developer console.

---

## ✨ Features

- **Interactive Developer Console** — Type commands like `whoami`, `skills`, `resume`, or `coffee` for a terminal-style experience
- **Dynamic 3D Project Cards** — Hover-activated 3D tilt with ambient light tracking and video previews
- **Deep Project Detail Views** — Architecture maps, live training charts, and animated data visualizations per project
- **Floating Portrait with Physics** — Organic holographic silhouette with cursor-reactive point lighting and chromatic aberration effects
- **Animated Particle Background** — Immersive interactive particle field using tsParticles
- **Smooth Scroll Animations** — Framer Motion powered reveal and parallax transitions throughout
